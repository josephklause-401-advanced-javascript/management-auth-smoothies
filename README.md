#### Running the app
Commands:
    "lint": "eslint .",
    "pretest": "npm run lint",
    "jest": "jest --runInBand",
    "test": "npm run jest -- --coverage",
    "test:watch": "npm run jest -- --watchAll",
    "test:verbose": "npm run test -- --verbose",
    "start": "node server.js",
    "start:watch": "nodemon server.js"

#### Tests
* Jest Tests: `npm test`
* Lint Tests: `npm run lint`
